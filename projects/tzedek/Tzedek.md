# Tzedek

> Israeli civic rights & volunteering ecosystem. 100% of revenue goes to donations.

## Status: Phase 1 — MVP (Volunteering Hub)

## Stack
- **Frontend:** Next.js 15 (App Router), Tailwind CSS
- **Backend:** Supabase (auth + DB + real-time)
- **Hosting:** Vercel
- **AI Layer:** Claude API or Gemini (TBD)

## Phases
1. **Volunteering Hub (MVP)** — Searchable/filterable org directory, Hebrew-first + English, admin panel, SEO-optimized pages
2. **Benefits Navigator** — AI matching engine (questionnaire -> eligible rights/services). Sources: Kol Zchut, gov.il, Bituach Leumi, MOD, HMOs. Multilingual (HE, EN, RU, AR). WhatsApp bot.
3. **IDF/Veterans Vertical** — Specialized layer for lone soldiers, injured soldiers, reservist families. Connects Phase 2 rights engine with Phase 1 volunteer orgs.
